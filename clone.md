# git clone(원격 저장소 복제)
- clone을 하면 원격 저장소와 자동으로 연결된 상태로 시작할 수 있다.
## clone이 하는것
- 원격 저장소의 모든 파일과 커밋 이력을 로컬로 복사
- 원격 저장소와의 연결 정보 자동 설정

# git commint(변경사항 기록)
- 로컬 저장소에 영구적으로 기록하는 명령어이다.
- push전까지 내 컴퓨터에 저장한다.

# git push(원격 저장소에 업로드)
- git commit한 기록들을 원격 저장소로 전송하는 명령어이다.

# git checkout(커밋 이력으로 되돌리기)
- checkout은 특정 시점의 코드를 확인하고 싶을때
- 가급적이면 변경사항 적지 말것
